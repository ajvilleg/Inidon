Note: Inidon requires Java. This is available at www.java.com.

Inidon v2.0
Purpose:

To calculate the frequency of each initiation codon found in every gene in a sequence
How to use:

This program reads files with a .FFN extension only. One source for these files is GenBank: ftp://ftp.ncbi.nih.gov/genbank/
For example, click on the link given above. Go into the genomes folder. Go into the Bacteria folder. Go into the Escherichia_coli_K12 folder and save the file U00096.ffn locally on your computer. Remember to take note of where you saved the file.
Click on the Choose .FFN file button and open the file you saved above. The resulting frequencies will be displayed. Clicking Output to Excel File will produce a CSV file at the same location as your FFN file.
